## 🚀 Overview

**Books API Analyzer** is a web-based project that fetches book data from the Google Books API and processes it through custom JavaScript functions.

The goal is to practice **JSON data handling**, **API integration**, and **modular JavaScript development**.

---

## 🧩 Project Structure
├── index.html # UI structure (provided)
├── script.js # API fetch logic (provided)
├── practica.js # Custom data analysis (my work)


---

## ⚙️ How It Works

- `script.js` fetches data from the **Google Books API**
- The response is returned as a **JSON object**
- The functions in `practica.js` analyze the data
- Results are rendered dynamically in the browser
